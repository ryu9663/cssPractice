# position


  relative안에서 absolute가 움직인다.
  
  ```CSS
  .login{
    
    position : relative;
    left:100px;
    
    background: url('url') center no-repeat;
    background-size: 60%;
}

.login_body{
    
    position : absolute;
    
    min-height:100vh;
    text-align: center;
    top:20%;
    left:0;
    right:0;
    
    
}
  ```
